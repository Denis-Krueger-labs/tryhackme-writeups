# Standard Methodology

The following methodology is consistently applied during room analysis.

## 1. Reconnaissance
- Network scanning
- Service identification
- Version enumeration

## 2. Service Enumeration
- Web directory discovery
- Credential testing (within lab scope)
- Manual inspection

## 3. Initial Access
- Identification of vulnerabilities
- Controlled exploitation

## 4. Privilege Escalation
- Local enumeration
- Misconfiguration analysis
- Exploitation of privilege boundaries

## 5. Defensive Considerations
- Indicators of compromise
- Log analysis perspective
- Hardening recommendations
